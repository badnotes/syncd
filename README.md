## 特性

- Go语言开发，编译简单、运行高效
- Web界面访问，交互友好
- 权限模型灵活自由
- 支持自定义构建
- 支持Git仓库
- 支持分支、Tag上线
- 部署Hook支持，可扩展性强
- 完善的上线工作流
- 邮件通知机制
